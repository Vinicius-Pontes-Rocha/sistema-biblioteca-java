 Sistema de Gerenciamento de Biblioteca 

Projeto desenvolvido em Java com foco em Programação Orientada a Objetos.
O sistema permite gerenciar livros e usuários de uma biblioteca através de um menu interativo no terminal.

* Cadastro de livros
* Cadastro de usuários
* Empréstimo de livros
* Devolução de livros
* Listagem de livros cadastrados
* Listagem de usuários cadastrados

 Tecnologias utilizadas

* Java
* Programação Orientada a Objetos (POO)
* ArrayList para armazenamento de dados
* Interface via terminal

 Estrutura do projeto

src
Main.java
Livro.java
Usuario.java


Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de Java, **POO e lógica de programação, além de servir como parte do meu portfólio de desenvolvimento.
